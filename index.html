Social Netwaork Analysis - Graph Visualizer

이 페이지는 컴퓨터 화면에서 올바르게 작동합니다

이 웹 페이지는 사회 관계망 데이터를 쉽게 시각화하고 분석할 수 있는 도구입니다.

각 페이지는 그래프 시각화, 집단 찾기, 연결 중심성(degree centrality) 및 고유벡터 중심성(eigenvector centrality) 계산을 지원합니다.

Sample Data

미리 저장되어 있는 예시 데이터를 활용할 수 있습니다

Table Data

페이지에서 직접 데이터를 입력하고 그래프를 그릴 수 있습니다

CSV Data

직접 제작한 csv 파일을 업로드하여 그래프를 그릴 수 있습니다
